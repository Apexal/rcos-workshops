Part 1: What is Heroku?

"Heroku is a cloud platform that lets companies build, deliver, monitor and scale apps — we're the fastest way to go from idea to URL, bypassing all those infrastructure headaches."
—Heroku Site
Let's check it out...

tl;dr

Instead of running and managing a server to try to run your webapps manually, Heroku lets you hand off the code to them and they handle all of the hard parts.

Part 2: Configuring App for Heroku

Here is our example Flask Python web sever:

                        
                            # server.py
                            from flask import Flask

                            app = Flask(__name__)

                            @app.route("/")
                            def hello_world():
                                return "<p>Hello, RCOS!</p>"

We run it locally with 

FLASK_APP=server flask run

What does Heroku need to know about the web app?

  • language automatically determined
  • dependencies (third-party packages)
  • start command
